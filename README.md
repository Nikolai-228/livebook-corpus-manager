# Живая книга ИМИ — Корпусный менеджер

Программный комплекс для хранения, разметки, поиска и визуализации материалов интернет-ресурса  
"Живая книга ИМИ-ИжГТУ имени М. Т. Калашникова".

## Команда
- **Николай** — парсинг Google Drive, загрузка данных в БД  
- **Егор** — лингвистическая разметка (леммы, NER, тематики, n-граммы)  
- **Тигран** — БД, API, корпусный менеджер  
- **Руслан** — интерфейс, визуализация

## Технологии
- БД: PostgreSQL  
- Бэкенд: Python (FastAPI)  
- Парсинг: Python + Google Drive API + BeautifulSoup  
- Лингвистика: pymorphy3, Natasha  
- Фронтенд: (уточнит Руслан)

## Структура проекта
- `backend/` — API и корпусный менеджер  
- `parser/` — скрипты для выгрузки данных с Google Drive  
- `linguistics/` — лингвистические модули  
- `frontend/` — пользовательский интерфейс  

## Запуск (будет дополняться)
1. Установить PostgreSQL, создать БД `livebook_corpus`
2. Выполнить `backend/migrations/init_db.sql`
3. Настроить ключ Google Drive API в `parser/config.py`
4. Запустить `python parser/drive_parser.py`
